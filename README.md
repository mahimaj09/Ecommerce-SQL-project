# Ecommerce-SQL-project

## 📌 Project Overview

This project demonstrates the design and implementation of an **E-commerce Database** using SQL.
It manages customer data, product inventory, orders, order details, and payment transactions.

## Database Structure

The project consists of the following tables:
* **Customers** – Stores customer details
* **Products** – Stores product information
* **Orders** – Contains order records
* **Order_Details** – Links products with orders
* **Payments** – Stores payment information

## Features
* Customer Management
* Product Inventory Tracking
* Order Processing System
* Payment Management
* Data Analysis using SQL Queries

## 🧠 SQL Concepts Used
* DDL (CREATE, ALTER)
* DML (INSERT, SELECT)
* Joins (INNER JOIN)
* Aggregate Functions (SUM, AVG, COUNT)
* GROUP BY & ORDER BY
* Subqueries

## 📊 Key Queries Implemented
* Total Sales Calculation
* Revenue by City
* Monthly Revenue
* Top 5 Customers by Spending
* Top Selling Products
* Order Status Summary
* Pending Payments
* Low Stock Products

## 📌 Conclusion

This project showcases how SQL can be used to efficiently manage and analyze an E-commerce system with real-world business insights.
